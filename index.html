What is CoffeeScript?

    CoffeeScript is a language that compiles into JavaScript. The golden rule of CoffeeScript is: “It is just JavaScript.”
  • Better JavasScript
  • Compiles to JavaScript
  • Less Code
  • Synatactic Sugar
    • CoffeeScript is an attempt to expose the good parts of JavaScript in a simple way
Difference between CoffeeScript vs Javascript
  • Missing syntax elements
  • Whitespaces
  • Best Practices

Functions are defined by an optional list of parameters in parentheses, an arrow, and the function body. The empty function looks like this: ->

CoffeeScript uses significant whitespace to delimit blocks of code.

Benefits of using CoffeeScript

  • Easier to Understand
  • Write less code while doing more
  • Reliable
  • Readable & maintainable
  • Class-based inheritance
  • Avoid problematic errors
  • Extensive library support
